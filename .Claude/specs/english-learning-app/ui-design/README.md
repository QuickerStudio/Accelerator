## 使用指南

1. **开始设计前**：先阅读 `01-design-system.md` 了解基础设计规范
2. **实现页面时**：参考对应的页面设计文档（02-04）
3. **使用组件时**：查看 `05-components.md` 了解可复用组件
4. **添加动画时**：参考 `06-animations.md` 保持动画一致性
5. **主题适配时**：查看 `07-themes-accessibility.md` 确保无障碍支持

## 设计原则

- **简洁明快**：清晰的视觉层次
- **卡片式布局**：使用圆角卡片组织内容
- **渐变色彩**：柔和的渐变背景
- **大字体层级**：清晰的标题和内容对比
- **充足留白**：保持界面呼吸感
