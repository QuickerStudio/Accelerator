# 设置页面导航界面 (Settings Pages)

本文件夹包含从"我的"页面进入的各个设置页面的详细设计文档。

## 文档列表

### 🔊 [voice-settings.md](voice-settings.md)
朗读音色设置页面
- 当前音色显示（渐变卡片）
- 推荐音色列表（美式/英式男声女声）
- 音色试听功能
- 语速调节（0.5x - 2.0x）
- 音量调节（0% - 100%）
- 状态管理与数据持久化

### 🤖 [model-settings.md](model-settings.md)
模型设置页面
- 当前模型显示（渐变卡片）
- 可用模型列表（GPT-4、GPT-3.5、Claude等）
- 模型参数调节：
  - 温度（Temperature）：0.0 - 2.0
  - 最大长度（Max Tokens）：256 - 4096
  - Top P：0.0 - 1.0
- 恢复默认设置功能
- 状态管理与数据持久化

### ⚙️ [general-settings.md](general-settings.md)
通用设置页面
- 通知设置（推送、邮件、声音、震动）
- 学习设置（每日目标、提醒时间、统计显示）
- 隐私与安全（修改密码、隐私设置、数据安全）
- 账号管理（绑定手机号、邮箱、第三方账号）
- 其他（关于、协议、更新、清除缓存）
- 退出登录功能
- 状态管理与数据持久化

## 设计规范

### 统一的顶部导航栏
所有设置页面都使用相同的顶部导航栏设计：
- 高度: 64.dp
- 背景: 深色 (#1E1E1E)
- 返回按钮: 48.dp × 48.dp, 左侧
- 页面标题: HeadlineMedium (20.sp), 白色

### 统一的卡片样式
- 圆角: CornerRadius.large (16.dp)
- 背景: 深色卡片 (#2A2A2A)
- 内边距: 16-20.dp
- 边距: 水平16.dp
- 阴影: elevation = 2-4.dp

### 统一的交互元素
- **开关**: 48.dp × 28.dp, 主题色
- **滑块**: 主题色, 带标签和当前值显示
- **按钮**: 圆角12.dp, 高度48.dp
- **箭头**: 20.dp, 灰色 (#94A3B8)

### 统一的动画规范
- 页面进入: 从右侧滑入, 300ms
- 开关切换: 滑动动画, 200ms
- 按钮点击: 缩放动画, spring效果
- 颜色过渡: tween, 200ms
