# Accelerator

英语学习加速器 - 一款基于 AI 的英语学习 Android 应用

## 项目简介

Accelerator 是一款帮助用户提升英语能力的移动应用，提供单词学习、口语训练、写作练习等功能。

## 技术栈

- **语言**: Kotlin
- **UI 框架**: Jetpack Compose
- **设计系统**: Material3
- **最低 SDK**: Android 8.0 (API 26)
- **目标 SDK**: Android 15 (API 35)

## 项目状态

🚧 项目正在开发中

## 开发环境

- Android Studio Ladybug | 2024.2.1 或更高版本
- JDK 17 或更高版本
- Gradle 8.9

## 构建项目

```bash
git clone https://github.com/QuickerStudio/Accelerator.git
cd Accelerator
./gradlew build
```
